# Contador de Truco Venezolano + VAR para los vivos

El Truco es el juego de cartas más famoso de Venezuela y, probablemente, el que más amistades ha puesto a prueba. Como siempre hace falta llevar la cuenta y las aplicaciones que hay por ahí suelen estar llenas de publicidad o diseños recargados, decidí programar este contador: un proyecto que nació para entretenerme y que, sobre todo, fuese funcional para el mundo real.

Como todos sabemos que en el truco venezolano la trampa y la vivesa son parte del reglamento no escrito, le añadí una función de historial o "log" de jugadas. Es básicamente el VAR de la mesa. Cada punto que se suma o se resta queda registrado con su total correspondiente, así que se acabó eso de "convenientemente" olvidarse de cuántos puntos se tenían tras un descuido.


### **🚀 Funcionalidades**
- **Estética Minimalista:** Diseño en negro puro para ahorrar batería y no cansar la vista en partidas nocturnas.
- **El VAR del Truco:** Registro detallado de cada movimiento para que nadie se pase de vivo con la cuenta.
- **Memoria Persistente:** Si cierras el navegador, te llaman o refrescas la página, los puntos y el historial siguen exactamente donde los dejaste.
- **Optimizado para Móvil:** Botones grandes y sin zooms accidentales para usarlo con una mano mientras sostienes las cartas con la otra.

### **🛠️ Tecnología**
- Este es un proyecto puramente Front-end desarrollado con:
- HTML5 y CSS3 para una interfaz limpia y oscura.
- JavaScript (Vanilla) para la lógica de puntos y el manejo de localStorage para la persistencia de datos.
- Un proyecto personal sencillo para que las partidas sean más fluidas y, sobre todo, para que nadie se coma puntos de más.
